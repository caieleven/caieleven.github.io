查看权限

使用命令：

可得到如下格式内容

内容含义是

权限 incode 所属用户 所属用户组 文件大小 修改时间 文件名
drwxrw-rwx 0 root root 1233 … temp

其中，各符号含义的具体含义如下：

  • d 是否为文件夹
  • r 是否有读取权限
  • w 是否有写入权限
  • x 是否有执行权限
  • -表示没有此项权限

权限具体含义：

  • 第一位表示是否为文件夹
  • 2-4位，表示所属用户的三项权限
  • 5-7位，表示所属用户组的三项权限
  • 8-10位表示其它用户的三项权限
