c++ cin详解

Posted on 2021-08-02 Edited on 2021-08-11 In c/c++

cin简介

​ cin是c++标准输入流对象，即istream类的对象。同理，cout是ostream类的对象。

​ 当我们从键盘中输入字符串的时候，需要敲回车键才能将这个字符串输送到缓冲区中，然后敲入的这个回车键(\r)会被转换为一个换行符(\n)。

​ cin读取数据时，也是从缓冲区中获取数据，缓冲区为空时，cin的成员函数阻塞，等待数据的到来，一旦缓冲区中有数据，就触发cin的成员函数去读取数据。

cin常用输入方法

cin >>

​ cin可以连续从键盘读取想要的数据，以空白字符作为i分隔符。

​ 当cin>>从缓冲区读取数据时，若缓冲区第一个字符是空白字符（比如空格、tab、换行等），cin会将其忽略并清楚，读取下一个字符，若缓冲区为空，则继续等待。

​ 若略过空白字符，也有方法，此处不展开。

cin.get()

  1. 读取一个字符

    1. 第一种用法的返回值是一个int值，成功则返回读取字符的ASCII码值
    2. 第二个的返回值为一个cin对象，因此支持链式操作，比如： cin.get(a).get(b)

  2. 读取一行

cin.getline()

​ 函数原型

​ 从函数原型中可知具体用法，因此省略。

​ * 注：cin.getline()不会将结束符残留在输入缓冲区，而cin.get()会。
